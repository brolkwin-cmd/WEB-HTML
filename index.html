COMPONENTES HTML / CSS

Curso de Desarrollo Web Full Stack BIT

HTML

HTML (HyperText Markup Language): Es la forma estándar de estructurar una página web y sus contenidos, sirviendo como base para definir los elementos que estarán dentro de la web, como pueden ser imágenes, textos, enlaces y otros tipos de media. Se usa por medio de etiquetas, y cada una de ellas permite una funcionalidad diferente.

css

CSS (Cascading Style Sheets): Es el lenguaje de diseño encargado de dar aspecto visual a la estructura creada en HTML. Permite definir colores, fuentes, tamaños y la apariencia general de los elementos, logrando que la interfaz sea atractiva y organizada para el usuario. Funciona mediante reglas que se aplican a las etiquetas existentes.

BOX MODEL

BOX MODEL (Modelo de Caja): Es el concepto que define que cada elemento en una web es una caja rectangular. Este modelo permite controlar el espacio ocupado por el contenido, el relleno interno (padding), el borde y el margen externo (margin), siendo fundamental para separar los elementos entre sí.

DISPLAY

Es la propiedad que determina cómo se debe mostrar un elemento en relación con los demás. Controla si un objeto debe ocupar todo el ancho disponible (bloque) o si puede compartir la misma línea con otros elementos (en línea), definiendo el flujo visual de la página.

FLEXBOX

Es un modelo de diseño avanzado que permite alinear y distribuir el espacio entre los elementos de forma dinámica y eficiente. Es ideal para crear diseños que se adapten a diferentes tamaños de pantalla, facilitando el centrado y la organización de grupos de objetos sin complicaciones.

MEDIDAS

Son las unidades utilizadas para definir el tamaño de los componentes, como el ancho, alto o tamaño de letra. Pueden ser fijas (como los píxeles) para un control exacto, o relativas (como los porcentajes) para que los elementos cambien de tamaño según el dispositivo donde se vean.

SELECTORES

Son los patrones que se utilizan para "apuntar" o elegir los elementos de HTML a los que queremos aplicar un estilo específico. Permiten que el código CSS sepa exactamente a qué parte de la estructura debe darle formato, ya sea a un grupo de elementos o a uno solo en particular.
